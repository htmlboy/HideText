Please notice: this demo will only work if your browser supports ligatures with spaces
(Chrome 17+, Safari 5+).

This is a special font with just one character (the rocket) linked to a paragraph as a ligature itself. So it's a real text block, hidden as a huge ligature if your browser supports ligatures with spaces (right now, only Chrome 17+ and Safari 5+).

As far as I've seen, Firefox supports ligatures but not ligatures with spaces.
